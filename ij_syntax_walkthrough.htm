This walk-through will help you activate auto-complete in IntelliJ.
When completed you should see something like this in your IDE:


In the left hand bar; Click "External Libraries" to select it.


Select "Project Structure" from the File Menu.


The "Project Structure" window appears.
Select "Libraries" in the list on the left.


From the toolbar, select the + icon.
(It is near the middle of the GUI.)


Select "New Lua Library"


Browse to the location of your "IntelliJ Auto-Complete Stubs"


IntelliJ wants to know which project to associate your new stubs library with. I used the default.


Select a name for your new stubs library.
I already had "G64" in use so here I am using "G63".


Walk through completed.
You should be able to see your API stubs listed by name in the left hand column where we began.
